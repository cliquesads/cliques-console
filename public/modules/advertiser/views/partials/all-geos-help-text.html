Geo Targeting & Blocking

By default, this Campaign will bid its Base Bid of {{ campaign.base_bid | currency : "$" : 2 }} for all available countries/regions/cities ("geos"). To change this, select any country or region on the map to the left, then select one of the following actions in the dialog box that follows:

Whitelist

Use the button to whitelist specific geos. Whitelisting geos means that only impressions originating from whitelisted geos will be bid for; impressions originating from anywhere else will be ignored.

When you whitelist a particular geo, you will, by definition, block all other geos. As such, if you have entered any blacklist settings, these settings will be cleared when you use the whitelist.

Blacklist

Use the button to blacklist (block) specific geos. Impressions originating from blacklisted geos will not be bid on in any capacity.

If you want to blacklist geos but already have a whitelist set, you may be asked to confirm these settings as your whitelist, by definition, blocks all other geos.

Customize Bids

Use the button to modify bids for specific geos. Modifying bids for a particular geo does NOT mean that your campaign will exclusively target this geo. To do this, you must also add this geo to your Whitelist (see above).
